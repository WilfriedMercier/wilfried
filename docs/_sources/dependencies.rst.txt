Required dependencies
=====================

Some modules require specific python libaries to be installed first. Below we list them by alphabetical order and give instructions as to how install them

+---------------------------------------------------+-----------------------------------------+----------------------------------------+---------------------------------------------------------------------------------------------------------------------+
| Dependency                                        | Install with pip                        | Install with conda                     | Other                                                                                                               |
+---------------------------------------------------+-----------------------------------------+----------------------------------------+---------------------------------------------------------------------------------------------------------------------+
| `astropy <https://https://www.astropy.org/>`_     | >>> pip install astropy                 | >>> conda install astropy              |                                                                                                                     |
+---------------------------------------------------+-----------------------------------------+----------------------------------------+---------------------------------------------------------------------------------------------------------------------+
| `colorama <https://github.com/tartley/colorama>`_ | >>> pip install colorama                | >>> conda install -c anaconda colorama |                                                                                                                     |
+---------------------------------------------------+-----------------------------------------+----------------------------------------+---------------------------------------------------------------------------------------------------------------------+
| `numpy <https://numpy.org/>`_                     | >>> pip install numpy                   | >>> conda install numpy                |                                                                                                                     |
+---------------------------------------------------+-----------------------------------------+----------------------------------------+---------------------------------------------------------------------------------------------------------------------+
| `matplotlib <https://matplotlib.org/>`_           | >>> python -m pip install -U matplotlib | >>> conda install matplotlib           | See `here <https://matplotlib.org/stable/devel/dependencies.html#optional-dependencies>`_ for optional dependencies |
+---------------------------------------------------+-----------------------------------------+----------------------------------------+---------------------------------------------------------------------------------------------------------------------+
