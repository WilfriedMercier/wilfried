Note

How it works

A set of threshold values are generated between the minimum and the maximum of the image, e.g.

  • [0, 1, 2] for a galaxy with a minimum of 0 and a maximum of 2 (using num=3)

These values are applied one after another onto the image as a minmum threshold, that is data points with value<threshold are masked. We get what we call a âsliceâ. For each slice, we compute its PA (angle starting from the vertical axis, counting anti clockwise).

  • If the âminmaxâ method is used, a galaxy with a PA close to 90Â° will not have a good PA estimation as different slices will have values oscillating around +90Â° and -90Â°, yielding a median value of approximately 0Â°â¦
