Code author: Hugo Plombat - LUPM & Wilfried Mercier - IRAP <wilfried.mercier@irap.omp.eu>

Utilties related to generating 2D mass and SFR maps using either LePhare or CIGALE SED fitting codes.

__init__(filters=[], files=[], errfiles=[], path='./', texpFactor=1, normalise=False, extData=None, extErr=None, **kwargs)[source]¶

Code author: Wilfried Mercier - IRAP <wilfried.mercier@irap.omp.eu>

Initialise SED objects.

Parameters

  • filters (list[str]) – (Optional) filters used to perform the SED fitting

  • files (list[str]) – (Optional) files corresponding to each filter (order is assumed similar)

  • errfiles (list[str]) – (Optional) error files corresponding to each filter (order is assumed similar)

  • path (str) – (Optional) common path appended before each file name

  • texpFactor (int or float) – (Optional) factor used to correct the exposition time when correcting the error maps

  • normalise (bool) – (Optional) whether to use the normalisation scheme or not

  • extData (list[int]) – (Optional) list of FITS file extensions for each data file. If None, 0th extension is used by default for each file.

  • extErr (list[int]) – (Optional) list of FITS file extensions for each error file. If None, 0th extension is used by default for each file.

Raises

TypeError –

  • if normalise is not a bool

  • if filters, files and errfiles are not all lists

  • if texpFactor is neither an int nor a float
